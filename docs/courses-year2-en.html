Course Content

The second-year courses focus on advanced knowledge of data science and network security, while enhancing students' practical skills.

Foundation Courses

Introduction to Networking

Course Description: Introduction to basic networking concepts, including TCP/IP protocols, network topology, and data transmission technologies.

Credits: 3 credits

Operating Systems

Course Description: Study of fundamental principles of operating systems, including process management, memory management, and file systems.

Credits: 3 credits

Web Design Basics

Course Description: Learning HTML, CSS, and JavaScript to design and build interactive websites.

Credits: 3 credits

Advanced Courses

Software Engineering

Course Description: Study of software development processes, including requirements analysis, design, implementation, and maintenance.

Credits: 3 credits

Data Analysis and Visualization

Course Description: Learning basic tools and techniques for data visualization to enhance data analysis capabilities.

Credits: 3 credits

Professional Courses

Basics of Network Security

Course Description: Learning about network attack and defense techniques, covering firewalls, intrusion detection systems, and more.

Credits: 3 credits

Business Data Analysis

Course Description: Analyzing business data and applying it to market forecasting, decision-making, and more.

Credits: 3 credits
