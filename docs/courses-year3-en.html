Course Content

The third-year courses focus on deepening professional skills and integrating practical operations.

Foundation Courses

Computer Organization

Course Description: Study of computer hardware architecture, including processors, memory, and I/O systems.

Credits: 3 credits

Mobile Application Development

Course Description: Introduction to the design and development of mobile applications, including app implementation.

Credits: 3 credits

Advanced Courses

Data Mining

Course Description: Study of data mining techniques, analyzing large-scale data, and extracting useful information.

Credits: 3 credits

Advanced Database Design

Course Description: In-depth study of database design and optimization, with implementation of database management systems.

Credits: 3 credits

Information Technology Project

Course Description: Students will undertake project implementation, applying what they have learned to real-world information management projects.

Credits: 3 credits

Professional Courses

Network Security Seminar

Course Description: Study of advanced topics in network security, with practical attack and defense exercises.

Credits: 3 credits

Cloud Computing Technology

Course Description: Study of cloud architecture and deployment, exploring big data applications in the cloud.

Credits: 3 credits
